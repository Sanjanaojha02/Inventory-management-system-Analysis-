# Inventory-management-system-Analysis-
Introduction :
This project presents an interactive  Inventory Management Dashboard developed using Tableau. The dashboard analyzes sales, warehouse, product, and customer data to provide a clear overview of business performance. It helps in monitoring revenue trends, profitability, and operational efficiency.

Objective :
1.To analyze total revenue and profit performance
2.To track monthly sales trends
3.To evaluate regional and warehouse performance
4.To identify top-performing products and categories

Tools :
1.Microsoft Excel
2.Tableau

Conclusion :
The dashboard provides a good understanding of inventory performance. It enables better decision-making by highlighting revenue trends, profitable products, warehouse efficiency, and potential customer risks.
